SQL training worldcup 2014
# SQL Practice: PJIN 75 Questions Solutions

本リポジトリは、PJIN（プログラミング実習場）で公開されている「SQL練習問題」全75問に対する解答コードを管理・記録するものです。

## 概要

SQLの基礎から実戦的な応用までを網羅的に学習することを目的とし、以下の2つのシリーズを完遂しました。

- **基礎力チェックシリーズ（第1回〜第50回）**:
    - SELECT文の基本、算術演算、集計関数、GROUP BY、HAVING、テーブル結合などの基本概念を網羅。
- **力試しシリーズ（第1回〜第25回）**:
    - 複雑な結合条件、サブクエリ（相関サブクエリを含む）、CASE式を用いた条件分岐、データの加工・整形などの実戦的クエリを網羅。

## 動作確認環境
### 解答の作成および動作検証に使用した環境は以下の通りです。

RDBMS:MySQL

学習を通じた習得スキル
本演習の完遂により、以下のSQL実装能力を体系的に習得しました。

1. データの抽出と集計
複数テーブルの外部結合（LEFT/RIGHT/FULL OUTER JOIN）を用いた整合性のあるデータ抽出

集計関数（SUM, AVG, COUNT, MAX, MIN）とGROUP BYによる多角的なデータ分析

2. 高度なクエリ制御
スカラーサブクエリおよび相関サブクエリを用いた動的なフィルタリング

CASE式によるカテゴリ分類およびフラグ立て

UNION、INTERSECT等の集合演算子を用いたデータ統合

3. データ品質と整合性
NULL値に対する適切なハンドリング（COALESCE関数の活用等）

複雑なWHERE句の条件定義による精密なデータ絞り込み
